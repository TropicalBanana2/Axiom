Axiom is a self-hosted scripting console for zombs.io. Persistent VPS bots, search-first in-game UI, and an opinionated scripting engine — all running on your own machine.

▶ Play now Manage sessions
three engines

Built around three independent runtimes.

UI engine

Schema-driven panel.

The in-game console reads a single JSON schema. Tabs, sections, controls — all data. Search anything across every tab with one keystroke.

scripting engine

ctx-based scripts.

Each control binds to a script. Compiled with new Function, sandboxed with try/catch. Access game globals through ctx.game — no fragile hooks.

session engine

Headless bots.

Spawn a Node bot per session. It keeps your character alive in zombs.io while you close the tab. Per-session protocol — every command carries an explicit sessionId.

why axiom

Banshee's capabilities. Cleaner architecture.

Persistent sessions

VPS bots that stay logged in. Reload the browser, reattach to the same character with the same state.

Search-first UI
